## Novidades

### 25/05/2020 - Input Ajax

```php
$form->text('cep', 'CEP', [
    'ajax' => [
        'url' => '/url/{parametro}',
        'fields' => [
            'endereco_rua' => 'logradouro',
            'endereco_bairro' => 'bairro',
            'cidade' => 'localidade',
            'uf' => 'uf'
        ],
        'method' => 'POST',
        'fieldsOptions' =>  [
            'uf' => 'uf_options'
        ]
        'data' => [
            'a' => 'b',
            'c' => 'd',
        ],
        'dataFields' => [
            'cep' => 'endereco_cep',
        ]
    ]
);
```
